# **App Name**: Vidyalaya Connect

## Core Features:

- User Authentication: Secure login for teachers, parents, students, and administrators.
- Notice Board: Display important notices and announcements to all users.
- User Role: Define Roles to authenticate users of teachers, parents, students, and administrators.

## Style Guidelines:

- Primary color: Dark Blue (#192A56) to reflect a professional and trustworthy environment.
- Background color: Light Pink (#F8D7E5) to create a welcoming and calm atmosphere.
- Accent color: Orange (#D35400) to highlight important information and CTAs.
- Body and headline font: 'PT Sans' for a balance of modern look and readability.
- Use simple, clear icons for navigation and user actions.
- Clean, card-based layout for easy access to information.
- Subtle transitions for loading and user interactions.
